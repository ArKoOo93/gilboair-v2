# חבילת מסירה נקייה — GilboAir

## מה כלול בחבילה
- קוד האתר המלא
- Dockerfile לפריסה
- README להתקנה והרצה
- קבצי public, server, src
- SEO בסיסי, favicon, OG image
- טופס + WhatsApp + עמוד תודה

## מה לא כלול
- קבצי טיוטה מיותרים
- קבצי עבודה זמניים
- גרסאות קוד ישנות

## מבנה חשוב
- `src/` — קוד האתר
- `public/` — תמונות, לוגו, favicon
- `server/` — שרת Express
- `data/` — שמירת פניות
- `Dockerfile` — פריסה ב־Coolify

## לפני מסירה/העלאה
- לבדוק שהמספרים והאימייל נכונים
- לבדוק טופס ו־WhatsApp
- לבדוק גלריה
- לבדוק תצוגת מובייל
