# נוסח הצעת מחיר — שדרוג אתר GilboAir

## הצעת מחיר לשדרוג ובניית אתר תדמיתי

הפרויקט כולל:
- אפיון וחידוד מבנה האתר
- עיצוב יוקרתי מותאם למיתוג הקיים
- בניית דף בית מכירתי
- עמודי אודות, שירותים, גלריה ויצירת קשר
- שילוב גלריית עבודות אמיתית
- טופס יצירת קשר עם חיבור ל־WhatsApp
- התאמה למובייל
- SEO בסיסי
- הכנה מלאה לעלייה לאוויר דרך שרת / Coolify / Git

### ערך הפרויקט
האתר החדש נבנה כדי:
- לשדר מקצועיות גבוהה יותר
- להציג את איכות הצילום בצורה מרשימה
- להקל על לקוחות להשאיר פנייה
- לייצר נוכחות דיגיטלית חזקה ומעודכנת

### נוסח אפשרי ללקוח
האתר החדש של GilboAir לא נבנה רק כדי “להיראות טוב”, אלא כדי להציג את העסק שלך ברמה שמתאימה לאיכות העבודה שלך.  
המטרה היא ליצור אתר שמרגיש מקצועי, יוקרתי, ברור ונוח, ושיגרום ללקוח להבין מהרגע הראשון שהוא נמצא מול עסק רציני.
